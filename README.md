# tdd-project
Aula Prática 1: Introdução a TDD, Testes Funcionais e Testes de Unidade Objetivo: Reproduzir uma sessão de desenvolvimento de software usando TDD (Test-Driven Development), com base no livro Test-Driven-Development with Python. Especificamente, iremos reproduzir os exemplos do Cap. 1 até o Cap. 3 do livro. Faça o exercício com o mindset de TDD.
